# Github Repo Sorter ↕

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app). It uses github API to fetch a users data and repositories and sorts it.

- Options for sorting are :
  - Date 📅
  - Forks 🔱
  - Stars ⭐

## 🏗 Build with

- ReactJS
- TailwindCSS
- Github API
